# CatBot

To run the project:

- Git and node.js must be installed  
- Run in command line:
    
        git clone git@github.com:fmiras/chatbot.git
        cd chatbot
        npm install
        npm start
        
- Go to http://localhost:3000/

To test the project:

        npm test
